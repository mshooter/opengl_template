# OpenGL Template written in Rust 
This is for my own learnign purposes. 

## How to run program 
```bash
git clone git@github.com:mshooter/opengl_template.git
cd opengl_template 
cargo build 
./target/debug/opengl_template 
OR 
cargo run
```
